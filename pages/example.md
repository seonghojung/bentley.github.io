# 첫 번째 게시글

이것은 GitHub Pages 정적 블로그의 예시 게시글입니다.

## 소개

이 블로그는 다음과 같은 기능을 제공합니다:

- 마크다운 기반 게시글 작성
- 다크/라이트 모드 지원
- 태그 및 카테고리 필터링
- 검색 기능
- 코드 하이라이팅
- Giscus 댓글 시스템

## 코드 예시

다음은 JavaScript 코드 예시입니다:

```javascript
// @ 예시 함수
function greet(name) {
  return "Hello, " + name + "!";
}

console.log(greet("World"));
```

## 마크다운 기능

이 블로그는 표준 마크다운 문법을 지원합니다:

- **볼드 텍스트**
- *이탤릭 텍스트*
- [링크](https://example.com)
- `인라인 코드`

### 리스트

1. 첫 번째 항목
2. 두 번째 항목
3. 세 번째 항목

### 인용문

> 이것은 인용문입니다.
> 여러 줄에 걸쳐 작성할 수 있습니다.

## 결론

이 블로그를 사용하여 마크다운으로 게시글을 작성하고 GitHub Pages에 배포할 수 있습니다.
